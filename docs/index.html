GSPO for Vietnamese Text-to-Speech Fine-tuning

Dinh Xuan Binh, Nguyen Duc Anh, & Team

Vietnamese Speech Research

📄 Code
Abstract. We present a reinforcement learning approach to fine-tune Vi-SparkTTS-0.5B, a Vietnamese large language model-based text-to-speech system, using Group Relative Policy Optimization (GSPO). Our method optimizes a multi-component reward function that jointly targets naturalness (UTMOS), intelligibility (WER), speaker similarity (SIM), and Vietnamese tone accuracy (VietTone), applied via LoRA adaptation over 2,800 training steps on 5,000 PhoAudiobook samples. The GSPO-trained model achieves improved naturalness scores (+2% UTMOS) while maintaining speaker similarity, demonstrating the viability of policy-gradient methods for TTS quality optimization.

1. Method

1.1 GSPO Algorithm

GSPO (Group Relative Policy Optimization) extends GRPO by computing per-group advantages across a batch of TTS completions. For each text prompt, the model generates N = 4 speech samples; a multi-metric reward is computed for each and the policy is updated to maximize expected reward while staying close to the reference model via a KL-divergence penalty:

J(θ) = E[r(y) − β · KL(πθ ‖ πref)]

1.2 Reward Function

The reward combines four components with the following weights:

  • SIM (30%) — Speaker similarity via ECAPA-TDNN + WavLM cosine similarity
  • UTMOS (40%) — Predicted naturalness MOS via UTMOS22-Strong
  • WER (20%) — Intelligibility measured by Whisper Large V3 (negated)
  • VietTone (10%) — Vietnamese tone accuracy via Unicode tone-mark comparison

1.3 Training Pipeline

Phase 1 Baseline Eval 100 samples
WER / SIM / UTMOS
Phase 2 Data Prep CSV → JSONL
BiCodec tokens
Phase 3 Reward Fn SIM+UTMOS
+WER+Tone
Phase 4 GSPO Train TRL · LoRA r=16
2×GPU · 2800 steps

2. Results

We evaluate on a held-out test set of 2,250 samples from PhoAudiobook. All metrics are computed using the same evaluation suite as baseline.

SFT (checkpoint-3000) 0.0628 2.9366 0.7768
SFT (final) 0.0497 2.9461 0.7765
GSPO-2800 (ours) 0.0363 ▼ stable 2.8984 ▲ +2% 0.6823 ≈

WER: Word Error Rate (Whisper Large V3). UTMOS: predicted MOS (UTMOS22-Strong). SIM: speaker cosine similarity (WavLM+ECAPA-TDNN). Highlighted row is our proposed method.
